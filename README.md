# Student Grade Calculator

This is my first Python web app built with Gradio.

## What the app does

The app allows a user to enter a student's scores for English, Mathematics, Science, and Computer. It then calculates:

- Total score
- Average score
- Grade
- Pass or fail result

## Tools used

- Python
- Gradio
- GitHub

## How to run the app

```bash
python -m pip install -r requirements.txt
python app.py
